# Changelog

All notable changes to this repository will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial repository structure
- Comprehensive README.md with phenomenon documentation
- api.json following Wyrdness schema v1.0.0
- SOURCES.md bibliography
- Issue templates for sighting reports, corrections, and source additions
- Pull request template

### Data Included
- Basic phenomenon metadata and classification (ENTITY_SPIRIT/Shadow Entities)
- Physical characteristics and behavioral patterns
- Geographic distribution (global, modern phenomenon)
- Historical timeline (ancient folklore, 1990s-present modern reports)
- Shadow types (Hat Man, Hooded Figure, Red-Eyed, Bedroom Watcher)
- Medical perspectives (sleep paralysis connection)
- Evidence catalog and encounter reports
- Theories (paranormal, scientific, psychological, skeptical)
- Sleep paralysis research and neurological explanations
- Cultural significance in internet age
- Investigation guidelines and resources
